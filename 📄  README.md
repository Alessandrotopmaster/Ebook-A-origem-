# O Caminho de Volta à Origem
**Acervo: Despertar – Biblioteca do Saber**

## Apresentação
Esta mensagem fala sobre o movimento do ser humano: como ele se afasta, se perde e, ao reconhecer seus erros, encontra de novo o caminho para a vida plena e eterna, por meio da Nova Aliança.

## Estrutura
- `index.html`: livro completo com capítulos e espaços para ilustrações
- `estilo.css`: formatação adaptada para celular e leitura confortável
- `imagens/`: pasta para adicionar as ilustrações de cada capítulo

## Como usar
Abra o arquivo `index.html` em qualquer navegador — não precisa instalar nada.
